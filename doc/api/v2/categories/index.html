ENDPOINT http://likecite.com/v2/categories

NOTE: New categories are created when users POST or PUT data to new or existing urls.

GET (xhtml)

This endpoints lists all available categories that have been created. This default representation displays the categories in xhtml format.

Usage Example:

Below is a "pretty-printed" representation of the returned xhtml.

<?xml version="1.0" encoding="utf-8"?>
<html xmlns="http://www.w3.org/1999/xhtml">
<head>
<title>Categories</title>
</head>
<body>
<h1>Categories</h1>
<div>
<a href="http://likecite.com/v2/categories/JAZ" class="category">JAZ</a>
</div>
<div>
<a href="http://likecite.com/v2/categories/News" class="category">News</a>
</div>
<div>
<a href="http://likecite.com/v2/categories/Sports" class="category">Sports</a>
</div>
</body>
</html>

GET (xml)

By appending a .xml to the endpoint specified above, one will recieve an XML formatted representation of the category data.

Usage Example:

Below is a "pretty-printed" representation of the returned xml.

<?xml version="1.0" encoding="utf-8"?>
<categories>
<category>
<uri>http://likecite.com/v2/categories/News</uri>
<name>News</name>
</category>
<category>
<uri>http://likecite.com/v2/categories/Sports</uri>
<name>Sports</name>
</category>
<category>
<uri>http://likecite.com/v2/categories/JAZ</uri>
<name>JAZ</name>
</category>
</categories>
