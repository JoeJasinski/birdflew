ENDPOINT http://likecite.com/v2/users/{email}

GET (xhtml)

This endpoint returns detailed information about the user given by the email url component. It presents the information in a clickable xhtml format.

Usage Example:

Below is a "pretty-printed" representation of the returned xhtml.

<?xml version="1.0" encoding="utf-8"?>
<html xmlns="http://www.w3.org/1999/xhtml">
<head>
<title>Users Detail</title>
</head>
<body>
<h1>Users Detail</h1>
<div class="user">
<p>
<a href="mailto:bob2@whitequail.org" rel="email">bob2@whitequail.org</a>
</p>
<p>
<a href="http://ec2-50-19-183-238.compute-1.amazonaws.com" rel="node">http://ec2-50-19-183-238.compute-1.amazonaws.com</a>
</p>
<p>
<a href="http://ec2-50-19-183-238.compute-1.amazonaws.com/v2/users/bob2@whitequail.org/urls" rel="urls">http://ec2-50-19-183-238.compute-1.amazonaws.com/v2/users/bob2@whitequail.org/urls</a>
</p>
</div>
</body>
</html>
